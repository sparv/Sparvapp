# SparvApp

This is SparvApp.

## Requirements

Following tools are needed to run the application
`node >= 8.4.0`

## Development

Before starting the first time, install all dependencies with:

`npm i`

To run the development server execute:

`npm run dev`

For running tests, execute: (not implemented yet)

`npm run test`
